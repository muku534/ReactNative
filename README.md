# ReactNative VS Code Extension

Create React Native projects with ease, just like Flutter! 🚀

## Features

- **Quick Start**: Press `Cmd+Shift+P` and type "React Native: New Project"
- **Templates**: Choose from 4 templates:
  - **New Application**: full RN app
  - **Bare Application**: minimal setup
  - **With Navigation**: React Navigation pre-configured
  - **New Library**: for npm package developers
- **Folder Picker**: Select where to create your project
- **Interactive Prompts**: All CLI questions handled via VS Code UI
- **Auto-terminal**: Runs the creation command automatically

## How to use

1. Open VS Code
2. Press `Cmd+Shift+P`
3. Type `React Native: New Project` and hit Enter
4. Follow the prompts!

## Development

1. Open this folder in VS Code
2. Run `npm install`
3. Press `F5` to launch the extension in a new window
4. Test the commands!
